# Syntaxe-AFA


# Commande de base
| Symbole / Action
| 0V Mets à 0 volt
| 5V Mets à 5 volts
| >> Pointeur vers actionneur/capteur
| {} Bascule (stockage 1 bit / état)
| & Boucle infinie


# Comparateur
| Symbole / Action
| G Comparateur (égal)
| GD Comparateur (différent)


# capteur / entrée
| Symbole / Action
| * Température
| % Puissance (courant/tension)
| ? Radar / distance / présence
| ! Front (changement d’état)
| TP Temps (temporisation)


# Horloge / contrôle système
| Symbole / Action
| ~ Cycle d’horloge
| ON Démarrage / reset
| OOF Arrêt complet


0V  5V  >>  {}  &  G  GD  *  %  ?  !  TP  ~  ON  OOF
